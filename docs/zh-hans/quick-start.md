TanStack Pacer 首先是一个框架无关 (framework-agnostic) 的工具库，主要用于实现速率限制 (rate limiting)、节流 (throttling)、防抖 (debouncing) 和队列 (queuing) 功能。

它可以通过我们的框架适配器 (framework adapters) 使用，也能直接在原生 JavaScript 或 TypeScript 中调用。
