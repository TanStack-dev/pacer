TanStack Pacer 首先是一個與框架無關 (framework-agnostic) 的實用程式庫，專注於速率限制 (rate limiting)、節流 (throttling)、防抖 (debouncing) 和佇列 (queuing) 功能。

它可搭配我們提供的任何框架適配器 (framework adapters) 使用，也能直接在原生 JavaScript 或 TypeScript 環境中運作。
